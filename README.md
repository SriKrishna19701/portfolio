This repository contains the source code for my personal portfolio website. It is designed with a cinematic, futuristic aesthetic to showcase my skills as a Full-Stack Developer, my project work, and my technical capabilities. The site is built with Next.js 15 and leverages modern animation libraries like GSAP and Framer Motion to create an immersive and interactive user experience.

## ✨ Key Features

*   **Cinematic Design:** A dark-themed UI with an "Electric Lime" accent, noise overlay, and subtle gradients to create a futuristic feel.
*   **Horizontal Project Showcase:** An engaging horizontal scrolling section powered by GSAP's ScrollTrigger to display featured projects.
*   **Advanced Animations:** Smooth animations and transitions are implemented using GSAP for complex timelines (like the hero text reveal) and Framer Motion for component-level interactions.
*   **Smooth Scrolling:** Integrated with Lenis for a fluid and seamless scrolling experience across the entire page.
*   **Custom Interactive Cursor:** A custom cursor that changes its appearance when hovering over interactive elements, enhancing user engagement.
*   **Responsive Layout:** The entire site is fully responsive, ensuring a consistent experience on all devices, from mobile phones to desktop monitors.

## 🚀 Tech Stack

*   **Framework:** [Next.js](https://nextjs.org/) 15
*   **Language:** [TypeScript](https://www.typescriptlang.org/)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/) 4
*   **Animations:**
    *   [GSAP (GreenSock Animation Platform)](https://gsap.com/) with ScrollTrigger for complex, scroll-based animations.
    *   [Framer Motion](https://www.framer.com/motion/) for UI animations and interactions.
*   **Smooth Scrolling:** [Lenis](https://lenis.studiofreight.com/)

## 📂 Project Sections

The portfolio is a single-page application divided into several distinct sections:

*   **`Hero`**: A full-screen introduction with an animated title and a brief professional summary.
*   **`ProjectWorks`**: A horizontal-scrolling section detailing key projects, each with a description, tech tags, and a link to the source code.
*   **`EngineeringCapabilities`**: A grid layout that provides an overview of my technical skills across different domains like Frontend, Backend, Mobile, and Cloud.
*   **`Experience`**: A summary of my academic background and qualifications.
*   **`Footer`**: A contact section with links to my Email, LinkedIn, and GitHub, along with a live-updating local time display.

## 🏃‍♂️ Running Locally

To run this project on your local machine, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/srikrishna19701/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies:**
    This project uses `npm` as the package manager.
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
